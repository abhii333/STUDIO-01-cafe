Boneyard capture harness

Renders each real component (from bone-fixtures.mjs) with its page's CSS inside breakpoint-sized iframes, runs the vendored snapshotBones(), and previews the resulting skeleton beside it. Click Download to save a component's .bones.json, then drop it into frontend/js/bones/ and commit.
Serve this file over HTTP (e.g. the site's dev server) — file:// blocks the fetch() used to load page CSS.
